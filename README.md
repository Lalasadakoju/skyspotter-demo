# ✈️ SkySpotter Alert

A frontend simulation project that mimics aircraft tracking near a user's location.

## 🚀 Features
- User input for name & location
- Simulated aircraft monitoring system
- Live countdown timer (ETA simulation)
- Dark/Light mode toggle
- Alert history tracking
- Statistics dashboard

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript (Vanilla)

## 📌 Note
This project uses simulated data (not real flight API). It was built to understand frontend logic, DOM manipulation, and UI interactions.
